# Юридические услуги - Личный юрист

Премиальный веб-сайт для предоставления юридических услуг.

## Особенности
- **Современный дизайн**: Темная тема с золотыми акцентами.
- **Адаптивность**: Полная поддержка мобильных устройств.
- **Направления практики**: Гражданское, Трудовое, Уголовное право и другие.
- **Обратная связь**: Форма для отправки заявок на почту.
- **Анимации**: Плавные переходы и интерактивные элементы.

## Технологии
- HTML5, CSS3 (Tailwind CSS)
- JavaScript (GSAP, Lenis, Lucide Icons)
- Node.js (для локального сервера)

## Как запустить локально
1. Клонируйте репозиторий.
2. Запустите `node server.js`.
3. Откройте `http://localhost:8000` в браузере.
